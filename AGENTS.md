## Workflow

- Use plan mode for non-trivial tasks
- Write plan in tasks/todo.md before coding
- Re-plan if needed
- Verify before marking complete

## Task Tracking

- Keep tasks/todo.md updated
- Mark progress
- Add short results per task

## Lessons

- Log mistakes in tasks/lessons.md
- Include cause + prevention

## Standards

- Keep solutions simple
- Minimize changes
- Fix root causes

## Verification

- Run tests
- Check behavior + edge cases
