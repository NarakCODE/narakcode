## Usage

```tsx
import { SpotifyPlaylistCard } from "@/components/spotify-playlist-card";
```

### Basic Usage

```tsx
export function SpotifyPlaylistDemo() {
  return (
    <SpotifyPlaylistCard
      playlistId="677KIyayzcVJeA77I9hEi0"
      title="My Awesome Playlist"
      description="The perfect soundtrack for coding"
    />
  );
}
```

### Custom Styling

```tsx
export function CustomSpotifyCard() {
  return (
    <SpotifyPlaylistCard
      playlistId="your-playlist-id"
      title="Chill Vibes"
      description="Relaxing music for focus and productivity"
      className="mx-auto max-w-2xl shadow-2xl"
    />
  );
}
```

## API Reference

### SpotifyPlaylistCardProps

| Prop          | Type     | Default                          | Description                               |
| ------------- | -------- | -------------------------------- | ----------------------------------------- |
| `playlistId`  | `string` | `"677KIyayzcVJeA77I9hEi0"`       | Spotify playlist ID from the playlist URL |
| `title`       | `string` | `"Spotify Playlist"`             | Display title for the playlist card       |
| `description` | `string` | `"Curated music for every mood"` | Description text shown below the title    |
| `className`   | `string` | `""`                             | Additional CSS classes for styling        |

### Getting Playlist ID

To get a Spotify playlist ID:

1. Open Spotify and navigate to your playlist
2. Click the three dots (⋯) next to the playlist name
3. Select "Share" → "Copy link to playlist"
4. Extract the ID from the URL: `https://open.spotify.com/playlist/[PLAYLIST_ID]`
