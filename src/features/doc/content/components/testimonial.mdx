## Usage

```tsx
import {
  Testimonial,
  TestimonialAuthor,
  TestimonialAuthorInfo,
  TestimonialAuthorName,
  TestimonialAuthorTagline,
  TestimonialAvatar,
  TestimonialAvatarImg,
  TestimonialAvatarRing,
  TestimonialQuote,
  TestimonialVerifiedBadge,
} from "@/components/testimonial"
```

```tsx
<DocTestimonial>
  <TestimonialQuote />

  <TestimonialAuthor>
    <TestimonialAvatar>
      <TestimonialAvatarImg />
      <TestimonialAvatarRing />
    </TestimonialAvatar>

    <TestimonialAuthorName>
      <TestimonialVerifiedBadge />
    </TestimonialAuthorName>

    <TestimonialAuthorTagline />
  </TestimonialAuthor>
</Testimonial>
```

## Composition

Use the following composition to build a `Testimonial`

```text
Testimonial
├── TestimonialQuote
└── TestimonialAuthor
    ├── TestimonialAvatar
    │   ├── TestimonialAvatarImg
    │   └── TestimonialAvatarRing
    ├── TestimonialAuthorName
    │   └── TestimonialVerifiedBadge
    └── TestimonialAuthorTagline
```
