# GPT-5.6 មាន 3 Models

![GPT-5.6 Sol Terra Luna](https://community.openai.com/uploads/short-url/yjRzKQdoybCZwXV7VWNz6dqZ4ff.jpeg?dl=1)

## 1. Sol

**Sol** ជា Flagship Model សម្រាប់ការងារធំៗ និងស្មុគស្មាញ។

- Performance ខ្លាំងបំផុត
- Advanced Reasoning
- Complex Coding
- Research
- Agent Workflow
- Production-grade Work

សាកសមសម្រាប់ Project ដែលត្រូវការ Accuracy ខ្ពស់ និង Workflow វែងៗ។

## 2. Terra

**Terra** ជា Balanced Model រវាង Performance, Speed និង Cost។

- Performance ល្អ
- ល្បឿនល្អ
- តម្លៃសមរម្យ
- សាកសមសម្រាប់ Everyday Work

Terra សាកសមសម្រាប់ការងារដែលត្រូវការសមត្ថភាពខ្លាំង ប៉ុន្តែមិនចង់ចំណាយខ្ពស់ដូច Sol។

## 3. Luna

**Luna** ជា Fast & Budget Model សម្រាប់ការងារប្រចាំថ្ងៃ។

- Response លឿន
- តម្លៃទាប
- Daily Tasks
- Chatbot
- Automation
- High-volume Requests

លេខ **5.6** សម្គាល់ Generation ខណៈ **Sol / Terra / Luna** សម្គាល់កម្រិត **Performance, Speed និង Cost**។

# GPT-5.6 ធ្វើអ្វីបាន?

GPT-5.6 ត្រូវបានរចនាសម្រាប់ **Agentic AI** និង **End-to-end Knowledge Work** ដែលអាច៖

- រៀបចំផែនការ
- ប្រើ Tools
- បែងចែកការងារ
- ដោះស្រាយ Multi-step Tasks
- គ្រប់គ្រង Workflow ស្មុគស្មាញ
- បង្កើត Artifacts ដូចជា Documents, Slides និង Spreadsheets

OpenAI ក៏បង្ហាញការរីកចម្រើនលើ **Software Engineering, Command Line, Computer Use, Scientific Research, Biology, Professional Knowledge និង Cybersecurity**។

# Sol, Max Reasoning និង Ultra Mode

**Sol** មាន Reasoning Setting កម្រិត **max** ដែលអាច “គិត” បានយូរជាងមុន សម្រាប់បញ្ហាពិបាកៗ។

ក៏មាន **Ultra Mode** ដែលប្រើ **Subagents** ដើម្បីបែងចែកការងារ និងដំណើរការច្រើនក្នុងពេលតែមួយ។

# API Pricing

![GPT-5.6 API Pricing](/images/blog/image.png)

| Model         | Input | Output |
| ------------- | ----- | ------ |
| GPT-5.6 Sol   | $5    | $30    |
| GPT-5.6 Terra | $2.50 | $15    |
| GPT-5.6 Luna  | $1    | $6     |

តម្លៃនេះគិតជា **per 1M tokens**។

GPT-5.6 ក៏មាន **Prompt Caching**៖

- Cache អប្បបរមា 30 នាទី
- Cache Write = 1.25x Input
- Cache Read បញ្ចុះតម្លៃដល់ 90%

សម្រាប់ App ដែលមាន Request ច្រើន វាអាចជួយកាត់បន្ថយ Cost បានច្រើន។
